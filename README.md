# Libraray-Management-System
A simple Library Management System developed using PHP and MySQL.
## Features
- Dashboard with library statistics
- Book Management
- User Management
- Book Issue & Return
- Transaction History
- Overdue Book Tracking
- Responsive Bootstrap Interface

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome

## Project Structure
```
index.php
books.php
users.php
transactions.php
config.php
style.css
database.sql
```

## Database Setup
1. Install XAMPP or WAMP.
2. Start Apache and MySQL.
3. Open phpMyAdmin.
4. Create a database named:

```
library_db
```

5. Import:

```
database.sql
```

6. Place the project inside:

```
htdocs/
```

7. Open your browser:

```
http://localhost/library-management-system-php/
```
